## 0. 분석 설계

- **메인 분석**: `v_cohort` = 공통 티어 코호트(PLATINUM+EMERALD+DIAMOND)
  - 15.1 9,543행 / 15.3 46,407행 — 양쪽 다 챔프 비교 충분
  - MASTER+ 22.9% vs 7.0% 같은 티어 분포 비대칭을 통제
- **대조**: 전체 풀 naïve view — "통제 안 하면 이렇게 왜곡된다" 사례용
- **신뢰성 가드**: 모든 드리프트 표에 `n_15_1` 컬럼 노출, 양쪽 패치 n<20 챔프는 메인 비교에서 제외(`LOW_N` 플래그), 픽률·승률 차이에 95% 정규근사 신뢰구간 부착
- **너프/버프 정의(`12_flagging`)**: `win_rate>=52%` AND 표본 충분(`pick_count>=30`)일 때
  - 픽률 상위 25%(NTILE=4) → 너프 (강력 + 인기)
  - 픽률 하위 25%(NTILE=1) → BUFF_HIDDEN_STRONG (강력 + 사장)

해석:

- **JUNGLE·MID·SUPPORT는 HHI ↓**(상위권 분산), **BOT은 약간 ↑**(여전히 쏠림 — 징크스/다이애나 등).
- 모든 포지션에서 **Gini ↑** — long tail이 길어졌다. 새로 등장한 챔프들이 대부분 낮은 픽률.
- **BOT은 다양성·평등성 둘 다 악화** (HHI ↑ + Gini ↑) — 메타가 좁아지는 유일한 라인.

## 5. 코호트 vs naïve 교란 효과 — `13_cohort_vs_naive_top` ⭐ 대시보드 핵심 패널

티어를 통제하면(코호트) vs 안 하면(naïve), 픽률이 오른 챔프 순위가 어떻게 흔들리는지.

### 5.1 대표 사례 — 이즈리얼

| | rank_cohort | rank_naive | pickrate_diff_pp |
|---|---|---|---|
| 이즈리얼 | **30위** | **5위** | 코호트 +0.13 / naïve +0.35 |

**naïve가 2.7배 과대평가**. 이즈리얼은 본래 고티어에서 픽률이 높은 챔프. 15.1(고티어 비중 큰 표본) → 15.3(중티어 비중 큰 표본) 전환에서 "마스터+ 비중 줄어든 효과"가 픽률 차에 그대로 들어옴. 같은 티어 구간(PLAT+EM+DIA)으로 좁히면 사실상 변화 없음.

→ **포트폴리오 한 줄**: *"이즈리얼은 naïve 분석에서 픽률이 오른 챔프 5위로 잡히지만, 티어를 통제하면 30위로 떨어진다. 픽률 상승의 대부분은 패치 효과가 아니라 표본의 티어 분포 차이였다."*

## 6. 너프/버프 후보 — `12_flagging_summary` (정의 조정 후)

### 6.1 정의가 왜 너프/버프에서 달라야 했나

초기 정의 (`pick_count >= 30 + win_rate >= 52% + NTILE quartile`)로 돌리면 **BUFF_HIDDEN_STRONG = 0건**이 나옴. 진단해보면:

```
patch=15.3, role=BOT, NTILE 분포:
  quartile 1: 22 챔프, pick_count 1~1
  quartile 2: 22 챔프, pick_count 1~4
  quartile 3: 22 챔프, pick_count 5~100
  quartile 4: 22 챔프, pick_count 109~1171
```

**quartile=1에 pick_count ≥ 15가 구조적으로 0건**. "사장됐으니 표본이 적다"는 정의의 본질을 컷 `>=30`이 자기모순으로 제거하고 있었다. NERF에는 표본도 자연히 따라오니 같은 컷을 적용해도 무방하지만, BUFF에는 별도 설계가 필요.

### 6.2 조정된 정의

| 라벨 | 인기 컷 | 표본 컷 | 강함 컷 |
|---|---|---|---|
| **너프** | `pop_quartile = 4` (top 25% 인기) | `pick_count >= 30` | `win_rate >= 52%` (점추정) |
| **BUFF_HIDDEN_STRONG** | `pop_quartile != 4` (비주류) | `pick_count >= 15` | **`Wilson 95% lower bound >= 50%`** (CI 하한) |

핵심 통찰: 표본이 작으면 점추정(평균 승률) 대신 **신뢰구간 하한**으로 강함을 정의한다. "표본은 작지만 95% 확률로 50%를 넘는다"는 보장이 곧 "표본 작음 + 진짜 강함"의 통계적 정의. SQL Wilson 공식: `( p + z²/(2n) − z·sqrt(p(1-p)/n + z²/(4n²)) ) / (1 + z²/n)`, z=1.96.

|---|---|---|---|
| 15.1 | TOP / BOT / SUPPORT | 9 / 6 / 6 | 0 / 0 / **1** (마오카이) |
| 15.1 | MID / JUNGLE | 7 / 5 | **1** (스웨인) / **1** (워윅) |
| 15.3 | TOP | 13 | **2** (아크샨, 케넨) |

총 10건. **롤 상식과 일치**: 카서스·브랜드 BOT, 스웨인 MID, 마오카이 SUP 같은 "비주류 픽으로 강한" 케이스가 정확히 잡힘.

### 6.4 분석 판단 메모

- NERF는 표본 컷 30 유지 — 인기 챔프 중 강캐를 보는 게 목적이라 임계를 낮출 필요 없음.
- BUFF는 정의 자체가 "사장된" — 컷이 그 정의를 죽이면 안 됨. **컷을 라벨별로 분리한 것이 분석가의 판단력.**
- 단순 `>=30`으로 0건을 두는 대신, "왜 0이 나왔는가?"를 통계적으로 분해해 컷을 재설계 → 면접 어필 포인트.

### 6.5 ⚠️ 규칙의 사각지대 — 잔나(잔나) 케이스

잔나(잔나) 통계 (코호트):
- 픽률 1.25% → 0.72% (**−0.52pp**, 픽률이 가장 많이 떨어진 챔프 (BOTTOM) 4위)
- 승률 45.4% → 56.7% (**+11.3pp**, CI [+0.94, +21.7] **진짜 신호**)

즉 **"픽률은 떨어지는데 승률은 오르는"** 패턴 — 직관적으로 "사장 중인 강캐"인데, 우리의 BUFF_HIDDEN_STRONG 룰에는 **잡히지 않았다**. 이유: 잔나 15.3 n=335로 **여전히 quartile 4(인기 톱25%)에 속함**. "비주류 = quartile≠4" 조건에 걸려서.

**규칙의 한계가 데이터로 드러난 사례다.** 현 규칙은 *픽률의 절대 수준(분위)* 만으로 "사장 여부"를 판단하는데, 잔나처럼 **"여전히 픽률은 높지만 떨어지고 있음 + 승률은 오름"** 같은 추세 기반 케이스는 포착하지 못한다.

### 6.6 개선 방향 — 다음 버전의 플래깅 규칙

- 정의에 **픽률 추세(Δpickrate)** 를 추가:
  - `FADING_STRONG` (새 라벨): `pick_count >= 30` + `pickrate_diff_pp < −0.3` + `winrate_diff_pp > 0 AND wr_ci_low > 0` (승률 상승이 통계적으로 보장)
  - 인기는 있지만 떨어지고 있고, 떨어지는 와중에 승률이 오르는 챔프 = "유저들이 떠나고 있지만 사실 더 강해진 챔프"
- 같은 논리로 **CLIMBING_WEAK**: pickrate ↑ + winrate ↓ + CI 보장 → "인기는 오르는데 진짜 약해진 챔프(FOTM 트랩)"
- 면접 답변 카드: *"이 분석을 더 발전시킨다면?" → "절대 픽률 분위 + 픽률 추세 + 승률 변화를 결합한 4축 라벨링. 잔나 같은 사각지대를 잡기 위해."*

| 퀸 | 교란 효과: naïve 11위 등장 | **W 스킬(고양된 감각) 공격속도 버프** | ⭐ 정확 |

### 9.2 ⭐⭐ 스웨인 케이스 — 분석 결정 → 발견 → 외부 검증의 완결 서사

이건 운이 아니다. PHASE 2 안에서 명시적으로 내린 분석 결정 하나가 **연쇄 효과로 PHASE 4에서 외부 검증을 받은 케이스**다. 인과 사슬을 통째로 풀면:

**① 첫 시도 결과**: 초기 버프 정의(`pick_count >= 30` + 인기 quartile=1)로 돌렸더니 **0건**.

**② 진단**: "사장됐으니 표본 적다"는 정의의 본질과 컷이 자기모순. 데이터로 확인 — quartile 1의 pick_count는 1~4뿐 (§6.1 표).

**③ 분석가 판단**: 버프 라벨에 한해 컷 분리 — `pick_count >= 15` + `pop_quartile != 4` + **Wilson 95% 신뢰구간 하한 ≥ 50%**.
> *"왜 컷을 라벨별로 분리했는가"의 답이 곧 면접 점수.*

**④ 발견**: 새 컷으로 10건의 버프 후보. 그 중 하나가 **스웨인 MID, n=21, win 81%, Wilson 하한 60%**.

**⑤ 외부 검증 (PHASE 4)**: 라이엇 15.3 공식 패치 노트에 **스웨인 버프 실재** (패시브 "탐욕의 무리" 영혼당 추가 체력 12 → 15, E 스킬 "굳건한 의지" 주문력 계수 70% 증가).

> **만약 ② 단계에서 0건을 그냥 비워뒀다면 ⑤ 외부 검증 카드를 통째로 날렸을 것**. 분석가의 컷 조정 결정이 곧 발견의 조건이었다.

→ 면접 카드 (한 호흡으로): *"버프 후보가 처음엔 0건이 나왔습니다. 컷이 정의를 자기모순적으로 제거하고 있었기 때문입니다. 컷을 라벨별로 분리해 Wilson 신뢰구간 하한으로 강함을 재정의했더니 스웨인이 잡혔고, 라이엇이 다음 패치에서 실제로 그 챔프를 버프했습니다. 분석가 판단이 곧 발견 조건이었다는 게 핵심입니다."*

### 9.3 매칭 안 된 신호 — 표본 노이즈 의심 확정

- **신 짜오 승률 +25.5pp** — 15.3 패치 노트에 신 짜오 변경 없음. **§3 신중 해석 박스의 의심 확정** → 표본 노이즈.
- **마스터 이, 트리스타나, 야스오** — 15.3 변경 없음. 표본 노이즈 확정.
- **초가스 승률 +18.3pp** — 직접 챔프 변경 없음. *"메타 흐름이지 챔프 자체 변화는 아님"* (탱커 친화 아이템·룬 변화 가능성).

이런 "매칭 안 됨"도 정직하게 기록하는 게 **신뢰의 핵심**. 분석은 "모든 신호가 다 진짜다"가 아니라 *어떤 신호가 진짜고 어떤 게 노이즈인지* 가린다.
