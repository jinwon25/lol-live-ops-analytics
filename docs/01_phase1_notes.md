## 의사결정 로그

- **패치 라벨**: `game_version` 앞 두 자리. 각 파일 주력 패치만 남겨 두 그룹을 깨끗하게 분리.
  - P153 5,453행(8%) 손실 — 15.1/15.2/15.4 등 잡음 패치 제거.
  - P151 19,691행(48.7%) 손실 — 큐 혼합 + 패치 외 패치 제거. 의도된 정제.
- **포지션**: MIDDLE→MID / BOTTOM→BOT / UTILITY→SUPPORT 캐노니컬 5종으로 통일.
- **팀 구분**: P151은 `team_id`(100/200)가 있어 명시적. P153은 명시적 팀 컬럼이 없어
  **(game_id, win) 그룹을 팀 대용**으로 사용 — 게임당 5명 win=1 / 5명 win=0으로 명확.
- **파생 지표**: KDA/GPM/DPM/KP 모두 두 패치 동일 공식으로 재계산 (원본 값이 있어도 무시).
- **solo_tier**: 양쪽 NaN 모두 `'UNKNOWN'` 으로 일관.

## ⚠️ 분석 신뢰성에 영향 — 솔로 티어 분포 비대칭

|  | 15.1 | 15.3 |
|---|---|---|
| 고티어 (M+) | **22.9%** | 7.0% |
| 다이아 | **26.2%** | 14.5% |
| 에메랄드+플래 | 19.9% | **59.4%** |
| 골드 이하 | 28.4% | 18.5% |

**해석**: 두 데이터셋은 단순한 "패치 차이"가 아니다. **수집된 티어 코호트 자체가 다르다.**
- 같은 챔피언이라도 다이아 메타와 에메랄드 메타는 픽 패턴이 다름.
- 픽률·승률 변화를 "패치 효과"로 단정하면 안 됨 → **샘플링 효과와 분리** 필요.

**PHASE 2 대응**:
1. 전체 풀 비교(naïve view)와 함께
2. **공통 티어 코호트(예: EMERALD~DIAMOND)에서 잘라낸 비교 뷰**를 별도로 제공.
3. 두 결과가 같은 방향이면 패치 효과로 봐도 무방하고, 다르면 샘플링 효과로 보고함.
